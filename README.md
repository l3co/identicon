# Identicon

Project to generate Identicon from token or string.

### Install Dependencies

Install dependencies from `mix`

```bash
mix deps.get
```

> if this not work

```bash
mix deps.compile
```

### Test

```bash
$ mix test
```

### Generate Docs

```bash
$ mix docs
```

### Create your images

```elixir
Identicon.main("elixir")
```

> result is on the same folder
